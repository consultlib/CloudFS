Getting Started¶

Application Developers¶

You can start off by checking out the Application Tutorial. If you want to write your own application, we have a skeleton application. available. You can easily fork this on GitHub, or just download the files. If you’re having problems with meta.json, you can read the packaging docs for help.

Theme Developers¶

There isn’t a theme tutorial yet, however, you can copy one of the existing themes, change the background images, and fiddle with the CSS files. See the packaging docs for information on how to make a package once you’re done.
